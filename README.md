# 🌤️ Weather App - Atmospheric Editorial

A premium, modern weather dashboard designed with high-end aesthetics, glassmorphism, and real-time data integration. This application provides a comprehensive atmospheric overview with a focus on visual excellence and user experience.



## ✨ Features

- **Real-time Weather Dashboard**: Instant updates on temperature, "feels like" metrics, and daily ranges.
- **Detailed Atmospheric Widgets**: Monitor Humidity, Wind Speed (and direction), Pressure, Visibility, and UV Index at a glance.
- **Air Quality Index (AQI)**: Detailed monitoring of atmospheric health.
- **Multi-Location Management**: Save and switch between different cities seamlessly.
- **Forecast & Maps**: Extended outlooks and geographical weather visualizations.
- **Firebase Authentication**: Secure user login and personalized data persistence.
- **Premium Design System**: Built with "Atmospheric Editorial" design tokens, featuring:
  - Custom Inter, Manrope, and Space Grotesk typography.
  - Interactive glassmorphism effects.
  - Smooth micro-animations and gradients.
  - Responsive layouts for all screen sizes.

## 🚀 Tech Stack

- **Core**: HTML5, Vanilla JavaScript (ES Module)
- **Styling**: Modern CSS3 (Glassmorphism, CSS Variables, Flex/Grid)
- **Backend/Auth**: [Firebase](https://firebase.google.com/)
- **Data Source**: OpenWeather API (via `app.js`)

## 🛠️ Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Sarthak-52/Weather-app.git
   cd Weather-app
   ```

2. **Configure Firebase:**
   Create a `firebase-config.js` file (or update the existing one) with your Firebase project credentials.

3. **API Keys:**
   Ensure your OpenWeather API key is configured in `app.js`.

4. **Run Locally:**
   Since this project uses ES Modules, it requires a local server. You can use:
   ```bash
   # Using Live Server (VS Code Extension) or:
   npx serve .
   ```
